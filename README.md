# モバナワ — ナワバトラー非公式シミュレーター

ナワバトラー（スプラトゥーン3）の非公式ファンアプリです。  
デッキ作成・ドローシミュレーション・試し置きモードを提供します。

## 免責事項

このアプリはナワバトラー（スプラトゥーン3）のファンが非公式に制作したものです。  
任天堂株式会社とは一切関係ありません。  
ナワバトラー・スプラトゥーンは任天堂株式会社の登録商標です。

---

## 開発

```bash
npm install
npm run dev
```

Built with React + TypeScript + Vite + Tailwind CSS.
